# Design Tokens

The complete v1.1.0 token surface, harvested from
[autara-customer-web/src/app/globals.css](https://github.com/augmara/autara-customer-web).
**These are the only colors, surfaces, and typography you should use** —
anything else is a drift indicator.

## Semantic surface stack

The primary API. Reference these as `var(--background)`, `var(--surface)`,
etc. — _not_ the underlying hex values.

<div className="my-6 grid grid-cols-2 gap-3 md:grid-cols-4">
  {[
    { name: "--background", value: "#FBFAF6", desc: "Warm off-white. Body." },
    { name: "--surface", value: "#FFFFFF", desc: "Cards, sheets, dialogs." },
    { name: "--surface-elevated", value: "#F4F2EC", desc: "Tinted skeletons, KPI containers." },
    { name: "--surface-warm", value: "#F7F5EE", desc: "Disabled input fill." },
  ].map(({ name, value, desc }) => (
    <div
      key={name}
      className="rounded-2xl border border-[var(--border-subtle)] p-4"
      style={{ background: value }}
    >
      <div className="text-[10px] font-semibold uppercase tracking-wider text-[var(--text-subtle)]">
        {name}
      </div>
      <div className="mt-1 text-sm font-bold tabular-nums text-[var(--text-strong)]">
        {value}
      </div>
      <div className="mt-2 text-xs text-[var(--text-muted)]">{desc}</div>
    </div>
  ))}
</div>

## Shadow tokens — all `none`

Autara's identity is hairline-edged. **No drop shadows.** All shadow
tokens evaluate to `none`. Consumers reading `var(--shadow-card)` get
the correct treatment automatically.

```css
--shadow-card: none;
--shadow-card-hover: none;
--shadow-cta: none;
--shadow-autara: none;       /* v1.0.x alias */
--shadow-autara-md: none;
--shadow-autara-lg: none;
```

Lift signal: a 1px border-color shift on hover (`--border-subtle` →
`--border-hover`). That's it. If you need more lift than that, the
component probably shouldn't lift.

## Typography

**Satoshi** is bundled inside the package. Consumers no longer need
`next/font/local` — just import the tokens.

<div className="my-6 space-y-4 rounded-2xl border border-[var(--border-subtle)] bg-[var(--surface)] p-6">
  <p className="text-5xl font-bold tracking-tight text-[var(--text-strong)]">
    Welcome to Autara
  </p>
  <p className="text-2xl font-medium text-[var(--text-strong)]">
    Australia's vehicle care marketplace
  </p>
  <p className="text-base text-[var(--text-muted)]">
    Book verified detailers, ceramic coaters, and tinters in minutes.
    Cancel free up to 24 hours before your slot.
  </p>
  <p className="text-sm text-[var(--text-subtle)]">
    Available weights: 400 (Regular), 500 (Medium), 700 (Black mapped).
  </p>
</div>

**Workspace rule** (from MEMORY policy): use **only** weights 400 / 500
/ 700. No w300 / w600 / w800 / w900. Light (300) is bundled for
completeness but production code should never reference it.
